**Название проекта:** Визуализирующий профилировщик для языка Rust

**Команда:** UraniumTeam

**Формат системы:** Desktop-приложение

**Цель проекта:**  
  Создать визуализирующий профилировщик для языка программирования Rust

**Описание:**  
  Программа собирает данные о выполнении тех или иных частей целевого приложения и визуализирует полученые даные для удобного анализа производительности целевого приложения

**Целевая аудитория:**  
  Разработчики высокопроизводительного ПО на языке Rust

**Основное преимущество:**  
  * Кроссплатформенность библиотеки профилирования
  * Возможность быстро и удобно получать визуальные данные

**Стек технологий:** Rust, C#, WPF

**Сценарий использования:**
  1) Разработчик подключает библиотуку профилирования к целевому приложению
  2) Компилирует и запускает данное приложение
  3) завершает работу целевого приложения и запускает визуализатор
  4) Указывает путь к log-файлу, созданному профилировщиком
  5) Как результат получает визуализированные данные профилировщика

**Основные требования к ПО для пользования:**  
  ОС:  
  * для запуска визуализатора: Windows 7 и новее + .NET Framework 4.7  
  * для использования библиотеки профилирования: любая из поддерживаемых Rust

**Порядок установки:**
  1) Подключить библиотеку профилировщика к целевой программе;
  2) Запустить визуализатор (установка визуализатора не требуется).
  
**Структура приложения в [alpha-релизе](https://github.com/AT-33/uranium-profiler/releases/tag/v0.1-alpha):**  
  /UraniumVisualizer.dll - подключаемая библиотека профилировщика  
  /WpfVisualizer.exe - приложение, визуализирующее данные, полученные библиотекой Profiler.rs
  /AvalonDock.Themes.VS2013.dll, /AvalonDock.dll - библиотеки необходимые для работы визуализатора
