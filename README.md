**Название проекта:** Визуализирующий профилировщик для языка Rust

**Команда:** UraniumTeam

**Формат системы:** Desktop-приложение

**Цель проекта:**
  Создать визуализирующий профилировщик для языка программирования Rust

**Описание:**
  Программа собирает данные о выполнении тех или иных частей целевого приложения и визуализирует полученые даные для удобного анализа производительности целевого приложения

**Целевая аудитория:**
  Компании, занимающиеся разработкой высокопроизводительного ПО на языке Rust

**Основное преимущество:**
  Возможность быстро и удобно получать визуальные данные

**Стек технологий:** Rust, C#, WPF

**Сценарий использования:**
  1) Разработчик подключает библиотуку профилирования
  2) Запускает визуалировщик
  3) Указывает целевую программу
  4) Как результат получает визуализированные данные профилировщика

**Основные требования к ПО для пользования:**  
  ОС:  
    для запуска визуализатора: Windows 7 и новее + .NET Framework 4.7  
    для использования библиотеки профилирования: любая из поддерживаемых Rust  
  Процессор: 1.2GHz  
  RAM: 4GB  

**Порядок установки:**
  1) Подключить библиотеку профилировщика;
  2) Запустить визуализатор (установка визуализатора не требуется).
  
**Структура приложения:**
  Profiler.rs - подключаемая библиотека профилировщика
  Vizualyzer.exe - приложение, визуализирующее данные, полученные библиотекой Profiler.rs
